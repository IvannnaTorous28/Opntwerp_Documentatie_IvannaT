Wat ging goed?

Tijdens deze week heb ik heel veel gewerkt aan zowel mijn website als mijn Figma-ontwerpen. Voor de website heb ik alles aangepast en alle informatie toegevoegd. In Figma heb ik mijn fouten verbeterd. Ik ben daar nog niet helemaal klaar mee, maar ik ben al bijna zover. Tijdens de les heb ik aan de leerkracht gevraagd hoe ik mijn website online kon zetten via GitHub en Netlify. Ik had een probleem met de mappenstructuur op GitHub. De leerkracht heeft een fout in mijn bestanden gevonden en mij daarbij geholpen. We hebben uiteindelijk een juiste mapstructuur gebruikt en mijn code daarin geplaatst. Daarna heeft de leerkracht mij geholpen om mijn website correct te koppelen aan Netlify. Nu staat mijn volledige website online en kan deze door iedereen bekeken worden.

Wat is de volgende stap?

Ik moet mijn ontwerp verder uitwerken en afronden.
